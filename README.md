This setup file installs the full NetProbe IoT Security Scanner on your Windows machine.

Key features in this release:

Installs the complete standalone security scanner application

Scans local networks to find and identify connected smart devices

Checks the National Vulnerability Database for known CVE risks

Runs active penetration tests to find weak passwords and open camera streams

Isolates compromised devices by blocking their network access at the firewall

Generates detailed PDF security reports automatically

Installation note: Download the setup executable, open it, and follow the prompts. Since this is a new security tool without a paid developer certificate, Windows Defender might show a warning screen. Click "More info" and "Run anyway" to proceed with the installation
