// workflow

How it works

Four automated stages run sequentially — from device discovery to actionable remediation.

🔍

Discover

ARP broadcast maps every device on your subnet. Double-verified to eliminate false positives.

🧠

Analyse

MAC vendor lookup, HTTP banner grab, TLS check, UPnP fingerprint, and OS detection via TTL.

🐛

Audit CVEs

Live NIST NVD query with retry logic. Full CVE descriptions, CVSS scores, and direct NVD links.

🔒

Remediate

Penetration test results, device isolation via Windows Firewall, and plain-English fix recommendations.

⚠️

Windows SmartScreen — Expected Behaviour

As an independent security tool without a corporate code-signing certificate, Windows SmartScreen may show a warning on first launch. This is normal for all new independent software.

To proceed: click "More info" → "Run anyway".
